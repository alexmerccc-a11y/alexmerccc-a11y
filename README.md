## 🚀 Featured Projects

### 🔗 SaaS Short Link System

基于 Spring Cloud Alibaba 构建的高并发 SaaS 短链接平台，支持短链接生成、跳转、分组管理、访问统计和风控能力。

**核心技术：**

`Spring Boot` · `Spring Cloud Alibaba` · `Redis` · `RocketMQ` ·  
`ShardingSphere` · `MySQL` · `Sentinel`

**项目亮点：**

- 使用布隆过滤器与缓存空值防止缓存穿透
- 使用分布式锁解决短链接并发创建问题
- 采用 RocketMQ 异步处理访问统计
- 使用 ShardingSphere 完成数据分片
- 通过 Sentinel 实现流量控制与服务保护
